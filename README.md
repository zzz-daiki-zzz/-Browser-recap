# -Browser-recap
GitHub Pagesだけで動作する、利用履歴トラッキングアプリです。
### PWA起動 Recap
- 起動回数・頻度を記録
- 朝型/夜型ユーザー判定

### ブラウザ滞在 Recap
- 1回あたりの利用時間を記録
- じっくり派/サッと派を判定

### 時間帯 Recap
- 24時間の利用パターンを可視化
- 活動ピーク時間を分析

### カレンダー型 Recap
- 利用日をカレンダーで表示
- 連続利用日数を計算


## 使い方
1. **サイトにアクセス** - 登録・ログイン不要
2. **（任意）PWAとしてインストール**
   - スマホ: 「ホーム画面に追加」
   - PC: アドレスバーの「インストール」
3. **普段は何もしなくてOK** - 開くだけで自動記録
4. **「Recapを見る」で分析結果を確認**

### データについて

-  保存場所: この端末のブラウザ内（localStorage）
-  サーバー送信なし
-  他人からは見えない

### データの削除

- アプリ内: 「データをリセット」ボタン
- ブラウザ設定: サイトデータの削除
